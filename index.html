How it Works

• Click on a tile to cycle it's token from 'x', 'o', or ' '
• With three tokens on a board with nine space, there are 39 possible combinations of game states, and only about 1400 valid gamestates with a winner.
In a valid & winning gamestate, the difference between played tokens must not be greater than 1.
• This program optimizes the runtime of the game logic required to check if a given board can be considered a valid winner by using a hash function which maps
each of the board spaces and the respective tokens played in that spot to a value between 0 and 39-1, summing each of the space's values
and using that integer as the index for the lookup table containing the validity of all the possible gamestates.
